# Roadmap

## In Progress

- [ ] Project creation — workspace organization per topic/project
- [ ] App integrations — connect third-party tools (Notion, GitHub, etc.)

## High Priority

- [ ] Video generation — generate video content via AI models
- [ ] Bring-your-own API key — let users supply their own provider keys
- [ ] Chat sharing — share a public read-only link to a conversation
- [ ] Team/workspace — invite members, shared chat history, shared credits

## Medium Priority

- [ ] Conversation search — full-text search across all chats
- [ ] Custom system prompt — per-project or per-chat system instructions
- [ ] Scheduled messages / reminders — send a prompt on a schedule
- [ ] Export chat — download conversation as Markdown, PDF, or JSON
- [ ] Model comparison — run the same prompt across multiple models side-by-side

## Low Priority

- [ ] Plugin / tool marketplace — community-built tools usable in chat
- [ ] Voice input & TTS output — speak to the assistant, hear responses
- [ ] Mobile app (PWA) — installable Progressive Web App
- [ ] Analytics dashboard — usage stats, credit burn-down, cost breakdown per model
- [ ] Webhook notifications — get notified on chat events via webhook

## Completed

- [x] Multi-provider LLM support — OpenAI, Anthropic, Google, Mistral, DeepSeek, xAI
- [x] Streaming chat responses — real-time SSE streaming with resumable streams
- [x] Authentication — email/password, Google OAuth, guest mode
- [x] Subscription plans — free, pro, max tiers with credit-based usage
- [x] Billing integration — Tripay payment gateway (monthly & yearly)
- [x] Credit ledger — track AI usage and remaining credits per cycle
- [x] Model selector — choose LLM per conversation from settings
- [x] File attachments — upload PDF, DOCX, XLSX, CSV, TXT per chat
- [x] Document ingestion — extract text, chunk, and embed documents for RAG
- [x] Vector search — semantic search over uploaded documents (pgvector)
- [x] S3 file storage — store attachments in S3-compatible object storage
- [x] Artifact generation — render code, text, image, and spreadsheet artifacts
- [x] Rich text editor — ProseMirror-based editor for text artifacts
- [x] Code editor — CodeMirror with syntax highlighting for code artifacts
- [x] Projects page — basic project listing UI
- [x] Settings page — user preferences and model management
- [x] Dark / light theme — system-aware theme toggle
- [x] Chat history — persist and browse past conversations
- [x] Message voting — thumbs up/down feedback on AI responses
